## --------INSTALLATION INSTRUCTIONS--------

Go to https://www.oracle.com/java/technologies/downloads/ and download the appropriate Java Development Kit for your operating system.
Navigate to your downloads folder and double-click the executable Java installer to install JDK.
Follow the JDK installation instructions.
Now, go to https://github.com/MrRedTomato/game_of_life and press the green "Code" button.
Then, select 'Download zip' to download a zip folder containing all the code to your disk.
Now, navigate to the zip file and right-click it. Select 'extract all' and select the folder you want to put all the code in.

# If you use windows:

Hit the Start menu and start typing 'cmd'. The 'Command Prompt' app should show up.
Run the command prompt app, then navigate to the directory with the downloaded files using the command:
```bash
  cd <"director name">
```
(Don't include the triangle brackets!)
Next, run the following commands:
```bash
  javac *.java
  java Main
```
# If you use mac:

Navigate to Applications/Utilities/Terminal.app to run the Terminal app.
Next, navigate to the folder where all the code is using the command:
```bash
  cd <"directory name">
```
(Don't include triangle brackets!)
Next, run the following commands:
```bash
  javac *.java
  java Main
```
